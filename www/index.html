BTCダッシュボードの見方（相場判断ガイド）
■ 1. Coincheck BTC価格（JPY）
・急に動いたら短期勢が動いているサイン
・直近の高値を超える → 上昇トレンド入りしやすい
・直近の安値を割る → 下落トレンド入りしやすい
買い得： 安値を割らずに反発した時 / 売り板が薄い時
売り得： 高値を超えられず反落した時 / 買い板が薄い時

■ 2. 変化率（%）
・±0.3% → 小さな波
・±1% → 明確な動き
・±3% → ニュース or クジラ級の変動
買い得： マイナスが縮小してプラスに転じた瞬間
売り得： プラスが縮小してマイナスに転じた瞬間

■ 3. 板情報（Orderbook）
・買い板が厚い → 下値が固い（落ちにくい）
・売り板が厚い → 上値が重い（上がりにくい）
買い得： 買い板が急に厚くなった時
売り得： 売り板が急に厚くなった時

■ 4. チャート（BTC/JPY）
・右肩上がり → 買いが強い
・右肩下がり → 売りが強い
買い得： 安値が切り上がっている / 押し目で反発
売り得： 高値が切り下がっている / 戻りで失速

■ 5. 半減期
・半減期前後はボラティリティが上がりやすい
買い得： 半減期の半年〜1年前の静かな時期
売り得： 半減期直後の過熱感が強い時

■ 6. クジラの動き
・大口送金量が増える → 相場が動く前兆
買い得： 大口送金量↑ ＋ 価格↑（買い集め）
売り得： 大口送金量↑ ＋ 価格↓（売り抜け）
